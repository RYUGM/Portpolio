사용법

0.첨에 RESET 버튼 누르면 엑셀파일 하나 만들어짐. 이게 있어야 돌아감

1.상품이름 입력 - 아무거나 알아볼수있게, 나중에 삭제할때 적어야함

2.사고싶은 상품 쿠팡에서 링크 복붙해넣기 - 쿠팡 말고 다른사이트는 안돌아감

3.원하는 가격 입력 - 숫자만 입력! , <- 이런거 절대 입력 금지 

4.위에 세개 입력하고 추가 -> 찾기 버튼 누르면 너가 입력한 가격보다 더 낮은제품 찾음(없으면 아무것도 안함 있으면 차액이 얼마인지 알려줌)

5.구입했거나 살생각 없어진 상품은 상품 이름에다 너가 정한 이름 입력하고 삭제 누르면 지워짐

RESET 버튼 누르면 기존 저장해둔 리스트 싹 다 날아가니 주의!
